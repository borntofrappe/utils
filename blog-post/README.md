## blog-post

Design for different elements meant to style a blog posts.

## global.css

Global stylesheet for all elements in the directory.

## blockquote

`<blockquote>` element. Add a colored border and if mask properties are supported replace the border with a custom visual.

## code

`<code>` element inline, `<code>` elements wrapped in code fences with a `<pre>` parent element. Inline, add a colored border and if mask properties are supported replace the border with a custom visual. Block, add rounded borders and absolute position the matching icon.

## post-rhythm

Vertical space for block-level elements. Follow the guidance of the tutorial [_improve the readability of the content on your website_](https://piccalil.li/tutorial/improve-the-readability-of-the-content-on-your-website/).

## links

Link elements with a particular class. Permalink, relative links, absolute links.
