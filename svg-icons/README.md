# svg-icons

Node script to create an object exporting the syntax from a series of SVG icons.

## script.js

The script considers SVG icons in the `./svg` directory and creates two files:

1. `icons.js`, the JavaScript file exporting an object with the icons. The name of the icons is used as a key, while the syntax in included as a value and as a string

2. `index.html`, a demo page to highlight the icons in a grid
